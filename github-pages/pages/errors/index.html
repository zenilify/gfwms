System Errors

SYS-001
Critical

Database Connection Failed

Unable to connect to the WMS database server

Symptoms:

  • Login screen shows "Connection Error"
  • Application hangs on startup
  • Data not saving or loading

Solutions:

  1. Check network connectivity
  2. Verify database server status
  3. Restart WMS services
  4. Contact system administrator

SYS-002
High

Memory Allocation Error

System running out of available memory

Symptoms:

  • Application freezes or crashes
  • Slow response times
  • "Out of memory" messages

Solutions:

  1. Close unnecessary applications
  2. Restart the WMS application
  3. Check system resources
  4. Contact IT support
View Details →
SYS-003
Medium

License Validation Failed

WMS license could not be validated

Symptoms:

  • Warning message on startup
  • Limited functionality
  • Periodic license prompts

Solutions:

  1. Check license file location
  2. Verify license expiration date
  3. Update license file
  4. Contact vendor support

User Errors

USR-001
Low

Invalid Login Credentials

Username or password is incorrect

Symptoms:

  • "Invalid username or password" message
  • Unable to access system
  • Login screen reappears

Solutions:

  1. Check caps lock status
  2. Verify username spelling
  3. Reset password if needed
  4. Contact system administrator
View Details →
USR-002
Medium

Insufficient Permissions

User does not have required access rights

Symptoms:

  • "Access denied" messages
  • Missing menu options
  • Cannot perform certain actions

Solutions:

  1. Check user role assignments
  2. Verify permission settings
  3. Request access from supervisor
  4. Contact system administrator
View Details →
USR-003
Low

Session Timeout

User session has expired due to inactivity

Symptoms:

  • Automatic logout
  • Redirected to login screen
  • Unsaved work lost

Solutions:

  1. Log back in to continue
  2. Save work frequently
  3. Adjust session timeout settings
  4. Stay active during work
