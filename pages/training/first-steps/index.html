Herzlich willkommen bei Georg Fischer! Dieses Modul führt Sie Schritt für Schritt durch Ihre ersten Erfahrungen mit dem Jungheinrich Warehouse Management System (WMS). Nach Abschluss dieses Moduls können Sie selbstständig mit dem System arbeiten. ## Lernziele Nach Abschluss dieses Moduls können Sie: - Sich erfolgreich am WMS anmelden und navigieren - Ihren Arbeitsplatz optimal einrichten - Grundlegende Systemfunktionen verwenden - Stammdaten verstehen und abfragen - Einfache Lagervorgänge durchführen - Hilfe und Support bei Problemen finden ## 1. Erste Anmeldung am System ### 1.1 Vorbereitung **Bevor Sie beginnen:** - ✅ Arbeitsplatz-PC ist eingeschaltet - ✅ Benutzername und temporäres Passwort erhalten - ✅ Arbeitsplatz-spezifische Informationen notiert - ✅ Diese Anleitung griffbereit ### 1.2 Schritt-für-Schritt Anmeldung #### **Schritt 1: WMS-Client starten** 1. **Desktop-Symbol "Jungheinrich WMS"** doppelklicken 2. **Anmeldefenster** öffnet sich automatisch 3. **Systemname** in der Titelleiste prüfen - "PRODUKTIV" für den Echtbetrieb - "TEST" für Übungen #### **Schritt 2: Anmeldedaten eingeben** 1. **Benutzername** eingeben (Login-Name) 2. **Passwort** eingeben (zunächst temporäres Passwort) 3. **"Anmelden"** klicken #### **Schritt 3: Passwort ändern (bei Erstanmeldung)** 1. **Passwort-Änderungsdialog** erscheint 2. **Neues Passwort** eingeben (mind. 8 Zeichen) 3. **Passwort bestätigen** 4. **Passwort-Regeln** beachten: - Groß- und Kleinbuchstaben - Mindestens eine Zahl - Mindestens ein Sonderzeichen - Nicht das alte Passwort #### **Schritt 4: Arbeitsbereich erkunden** 1. **Hauptfenster** öffnet sich 2. **Navigationsbereich** links erkunden 3. **Arbeitsbereich-Tabs** oben bemerken 4. **Benutzerinformationen** rechts oben prüfen ### 1.3 Erste Orientierung #### **Bildschirmaufbau verstehen** ``` ┌─────────────────────────────────────────────────────┐ │ Jungheinrich WMS - PRODUKTIV [Benutzer] [X]│ ├─────────────────────────────────────────────────────┤ │ [Arbeitsbereich 1] [Arbeitsbereich 2] [3] [4] │ ├─────────────┬───────────────────────────────────────┤ │ Navigation │ Inhaltsbereich │ │ - Favoriten │ ┌─────────────────────────────────────┐│ │ - Suche │ │ Breadcrumb: Start > ... ││ │ - Menüs │ │ ││ │ │ │ Hauptinhalt ││ │ │ │ ││ │ │ └─────────────────────────────────────┘│ └─────────────┴───────────────────────────────────────┘ ``` #### **Wichtige Bereiche identifizieren** - **Suchfeld** für schnelle Navigation - **Favoriten** für häufig genutzte Funktionen - **Hauptmenü** für alle Systemfunktionen - **Breadcrumb** für aktuelle Position ## 2. Arbeitsplatz einrichten ### 2.1 Individuelle Anpassungen #### **Favoriten erstellen** 1. **Häufig genutzte Funktion** öffnen 2. **Stern-Symbol** in der Menüleiste klicken 3. **Favorit** wird gespeichert 4. **Favoriten-Reihenfolge** per Drag & Drop ändern #### **Arbeitsbereich-Organisation** 1. **Arbeitsbereich 1**: Tägliche Aufgaben 2. **Arbeitsbereich 2**: Spezielle Funktionen 3. **Arbeitsbereich 3**: Berichte und Abfragen 4. **Arbeitsbereich 4**: Hilfe und Dokumentation ### 2.2 Arbeitsplatz-spezifische Einstellungen #### **Drucker-Konfiguration** 1. **System** > **Drucker** aufrufen 2. **Standard-Drucker** für Ihren Arbeitsplatz wählen 3. **Etiketten-Drucker** testen 4. **Drucker-Eigenschaften** prüfen #### **Waagen-Konfiguration** (falls vorhanden) 1. **Waagen-Verbindung** testen 2. **Kalibrierung** prüfen 3. **Tara-Werte** einstellen 4. **Genauigkeit** kontrollieren #### **Barcode-Scanner** (falls vorhanden) 1. **Scanner-Verbindung** testen 2. **Scan-Modi** einstellen 3. **Beep-Einstellungen** anpassen 4. **Testscans** durchführen ### 2.3 Ergonomie und Sicherheit #### **Bildschirm-Einstellungen** - **Helligkeit** optimal einstellen - **Kontrast** für gute Lesbarkeit - **Schriftgröße** anpassen - **Bildschirmposition** für Nacken-Schonung #### **Arbeitsplatz-Sicherheit** - **Nottaste** lokalisieren - **Fluchtweg** merken - **Erste-Hilfe-Ausrüstung** finden - **Brandschutz-Einrichtungen** identifizieren ## 3. Grundlegende Systemfunktionen ### 3.1 Navigation beherrschen #### **Menü-Navigation** 1. **Hauptmenü** expandieren/minimieren 2. **Untermenüs** aufklappen 3. **Menüpunkte** auswählen 4. **Zurück-Navigation** mit Breadcrumb #### **Suchfunktion nutzen** 1. **[Strg] + F** für Suchfeld 2. **Menüpunkt-Name** eintippen 3. **Vorschläge** auswählen 4. **Enter** für Bestätigung #### **Tastenkombinationen** - **[Strg] + M**: Menü ein-/ausblenden - **[Strg] + 1-4**: Arbeitsbereich wechseln - **[F1]**: Hilfe aufrufen - **[F5]**: Ansicht aktualisieren - **[Esc]**: Abbrechen/Zurück ### 3.2 Daten-Filterung und -Suche #### **Basis-Filter verwenden** 1. **Tabelle** öffnen (z.B. Artikelübersicht) 2. **Filter-Symbole** in Spaltenköpfen nutzen 3. **Suchkriterien** eingeben 4. **Filter anwenden** und Ergebnisse prüfen #### **Erweiterte Suche** 1. **"Erweiterte Suche"** öffnen 2. **Mehrere Kriterien** kombinieren 3. **Datumsfilter** für Zeiträume 4. **Saved Searches** für wiederholte Suchen ### 3.3 Daten exportieren #### **Einfacher Export** 1. **Tabelle** markieren 2. **Rechtsklick** > **Exportieren** 3. **Excel-Format** wählen 4. **Datei** speichern #### **Berichte generieren** 1. **Berichte** > **Standardberichte** 2. **Report** auswählen 3. **Parameter** eingeben 4. **Bericht** generieren und speichern ## 4. Stammdaten verstehen ### 4.1 Artikel-Stammdaten #### **Artikel suchen** 1. **Stammdaten** > **Artikelstamm** 2. **Artikelnummer** oder **Beschreibung** eingeben 3. **Suche** starten 4. **Artikel** auswählen #### **Artikel-Informationen verstehen** - **Artikelnummer**: Eindeutige Identifikation - **Beschreibung**: Artikel-Bezeichnung - **Verpackungseinheit**: Anzahl pro Einheit - **Gewicht/Volumen**: Physikalische Eigenschaften - **ABC-Kennzeichen**: Wichtigkeits-Klassifizierung - **FIFO-Kennzeichen**: First-In-First-Out #### **Artikel-Details anzeigen** 1. **Artikel** doppelklicken 2. **Registerkarten** durchsehen: - **Grunddaten** - **Verpackung** - **Lagerung** - **Bilder** - **Zusatztext** ### 4.2 Ladehilfsmittel (LHM) #### **LHM-Typen verstehen** - **Paletten**: Für HRL (Hochregallager) - **Behälter**: Für AKL (Automatikkleinteilelager) - **Spezial-LHM**: Für besondere Anforderungen #### **LHM-Nummer-System** - **Aufbau**: Präfix + laufende Nummer - **Barcode**: Für automatische Erkennung - **Etikettierung**: Eindeutige Kennzeichnung #### **LHM-Status prüfen** 1. **Stammdaten** > **LHM-Verwaltung** 2. **LHM-Nummer** eingeben 3. **Status** prüfen: - **Aktiv**: In Verwendung - **Leer**: Verfügbar - **Gesperrt**: Nicht verwendbar - **Wartung**: In Reparatur ### 4.3 Lagerplätze und Koordinaten #### **Lagerort-System verstehen** - **Aufbau**: Bereich-Gang-Ebene-Platz - **Beispiel**: HRL-01-02-03 (HRL, Gang 1, Ebene 2, Platz 3) - **Etikettierung**: Physische Beschriftung #### **Lagerort-Informationen abrufen** 1. **Lagermodellierung** > **Lagerplätze** 2. **Koordinaten** eingeben 3. **Platz-Details** anzeigen: - **Abmessungen** - **Gewichtslimit** - **Aktueller Bestand** - **Reservierung** ## 5. Einfache Lagervorgänge ### 5.1 Bestandsabfrage #### **Artikel-Bestand prüfen** 1. **Bestand** > **Bestandsabfrage** 2. **Artikelnummer** eingeben 3. **Bestände** anzeigen: - **Verfügbar**: Für Auslieferung - **Reserviert**: Für Aufträge - **Gesperrt**: Qualitätsprüfung - **Gesamt**: Alle Bestände #### **Lagerplatz-Bestand prüfen** 1. **Lagerplatz-Koordinaten** eingeben 2. **Belegung** anzeigen 3. **Artikel-Details** prüfen 4. **Mengenwerte** kontrollieren ### 5.2 LHM-Verfolgung #### **LHM-Position finden** 1. **Lagermodellierung** > **LHM-Suche** 2. **LHM-Nummer** eingeben 3. **Aktuelle Position** anzeigen 4. **Bewegungshistorie** prüfen #### **LHM-Bewegung verfolgen** 1. **Transport** > **Transport-Übersicht** 2. **LHM-Nummer** filtern 3. **Aktuelle Transporte** anzeigen 4. **Status** überwachen ### 5.3 Einfache Buchungen #### **Bestand korrigieren** (mit Berechtigung) 1. **Bestand** > **Bestandskorrektur** 2. **Artikel** und **LHM** auswählen 3. **Neue Menge** eingeben 4. **Grund** für Korrektur angeben 5. **Korrektur** bestätigen #### **Umlagerung erstellen** (mit Berechtigung) 1. **Umlagerung** > **Umlagerung erstellen** 2. **Quell-LHM** auswählen 3. **Ziel-Lagerplatz** eingeben 4. **Umlagerung** freigeben ## 6. Hilfe und Support ### 6.1 Integrierte Hilfe #### **Hilfe-System nutzen** 1. **[F1]** drücken für Kontext-Hilfe 2. **Hilfe** > **Benutzerhandbuch** 3. **Suchfunktion** in der Hilfe 4. **Themen-Index** durchsuchen #### **Tooltips und Hinweise** - **Maus** über Symbole für Erklärungen - **Statusleiste** für aktuelle Informationen - **Fehlermeldungen** aufmerksam lesen - **Warnungen** ernst nehmen ### 6.2 Support-Kanäle #### **Interne Unterstützung** 1. **Schichtführer** bei operativen Fragen 2. **Kollegen** für Erfahrungsaustausch 3. **Trainer** für Schulungsfragen 4. **Superuser** für erweiterte Funktionen #### **Technischer Support** - **ServiceDesk**: +41 81 770 5678 - **E-Mail**: wms-support@georgfischer.com - **Intranet**: WMS-Portal > Support - **Notfall-Hotline**: 24/7 verfügbar ### 6.3 Selbsthilfe-Ressourcen #### **Dokumentation** - **Benutzerhandbuch**: Vollständige Anleitung - **Quick Reference**: Kurze Übersichten - **FAQ**: Häufige Fragen und Antworten - **Video-Tutorials**: Schritt-für-Schritt Anleitungen #### **Training-Materialien** - **Online-Kurse**: Vertiefende Schulungen - **Webinare**: Aktuelle Themen - **Praxis-Workshops**: Hands-on Erfahrung - **Zertifizierung**: Kompetenz-Nachweis ## 7. Praktische Übungen ### Übung 1: Erste Anmeldung 1. **Melden Sie sich am WMS an** 2. **Ändern Sie Ihr Passwort** 3. **Erkunden Sie die Oberfläche** 4. **Erstellen Sie erste Favoriten** ### Übung 2: Navigation trainieren 1. **Navigieren Sie zu "Stammdaten > Artikelstamm"** 2. **Suchen Sie einen Artikel** 3. **Verwenden Sie verschiedene Filter** 4. **Exportieren Sie die Suchergebnisse** ### Übung 3: Bestandsabfrage 1. **Öffnen Sie die Bestandsabfrage** 2. **Suchen Sie einen Artikel** 3. **Prüfen Sie die Bestände** 4. **Notieren Sie die Lagerplätze** ### Übung 4: LHM-Verfolgung 1. **Suchen Sie ein LHM** 2. **Prüfen Sie den aktuellen Standort** 3. **Schauen Sie die Bewegungshistorie an** 4. **Verstehen Sie den Status** ### Übung 5: Arbeitsplatz-Einrichtung 1. **Konfigurieren Sie Ihre Favoriten** 2. **Organisieren Sie Ihre Arbeitsbereiche** 3. **Testen Sie Drucker und Scanner** 4. **Speichern Sie Ihre Einstellungen** ## 8. Häufige Anfängerfehler ### 8.1 Navigationsfehler #### **Problem: Verloren im Menü** - **Lösung**: Breadcrumb-Navigation nutzen - **Tipp**: Favoriten für häufige Ziele erstellen - **Hilfe**: Suchfunktion verwenden #### **Problem: Falsche Funktion geöffnet** - **Lösung**: [Esc] drücken oder Schließen-Button - **Tipp**: Menünamen genau lesen - **Hilfe**: Tooltips beachten ### 8.2 Dateneingabe-Fehler #### **Problem: Falsche Artikel-Nummer** - **Lösung**: Artikel-Suche verwenden - **Tipp**: Barcode-Scanner nutzen - **Hilfe**: Artikel-Beschreibung als Suchkriterium #### **Problem: Ungültige Lagerplätze** - **Lösung**: Lagerplatz-Abfrage verwenden - **Tipp**: Koordinaten-System verstehen - **Hilfe**: Etiketten am Lagerplatz prüfen ### 8.3 System-Verständnis #### **Problem: Berechtigungsfehler** - **Lösung**: Schichtführer informieren - **Tipp**: Nur erlaubte Funktionen verwenden - **Hilfe**: Benutzergruppe prüfen lassen #### **Problem: Daten nicht gespeichert** - **Lösung**: Speichern-Button verwenden - **Tipp**: Pflichtfelder ausfüllen - **Hilfe**: Fehlermeldungen beachten ## 9. Prüfung und Zertifizierung ### 9.1 Lernerfolgskontrolle #### **Grundlagen-Quiz** 1. **Wie melden Sie sich am WMS an?** 2. **Welche Tastenkombination öffnet die Suche?** 3. **Was bedeutet das ABC-Kennzeichen?** 4. **Wie erstellen Sie einen Favoriten?** 5. **Wo finden Sie Hilfe im System?** #### **Praktische Aufgaben** 1. **Artikel-Suche** mit verschiedenen Kriterien 2. **Bestandsabfrage** für mehrere Artikel 3. **LHM-Verfolgung** mit Statusprüfung 4. **Export** von Suchergebnissen 5. **Arbeitsplatz-Konfiguration** vollständig ### 9.2 Bewertungskriterien #### **Bestanden (80% oder mehr)** - ✅ Selbstständige Anmeldung - ✅ Sichere Navigation - ✅ Erfolgreiche Datenabfrage - ✅ Korrekte Bedienung - ✅ Hilfe-Nutzung #### **Nachschulung erforderlich (unter 80%)** - ❌ Anmelde-Probleme - ❌ Navigation unsicher - ❌ Fehler bei Datenabfrage - ❌ Bedienung fehlerhaft - ❌ Hilfe nicht genutzt ### 9.3 Zertifikat Nach erfolgreichem Abschluss erhalten Sie: - **"WMS Erste Schritte" Zertifikat** - **Berechtigung** für Grundfunktionen - **Zugang** zu weiterführenden Modulen - **Eintrag** in Ihr Schulungsportfolio ## 10. Nächste Schritte ### 10.1 Weiterführende Module Nach diesem Grundkurs empfehlen wir: 1. **Wareneingang** (Empfohlen) - Wareneingangs-Prozesse - Buchungen und Kontrollen - Qualitätsprüfung 2. **Bestandsverwaltung** (Empfohlen) - Inventur-Verfahren - Bestandskorrekturen - Umlagerungen 3. **Kommissionierung** (Je nach Rolle) - Picklisten-Bearbeitung - Kommissionier-Strategien - Arbeitsplatz-Optimierung 4. **Berichtswesen** (Für Führungskräfte) - Standard-Berichte - Dashboard-Konfiguration - Kennzahlen-Analyse ### 10.2 Praktische Erfahrung #### **Begleitung durch erfahrene Kollegen** - **Erste Woche**: Intensive Betreuung - **Zweite Woche**: Selbstständige Arbeit mit Unterstützung - **Dritte Woche**: Vollständige Selbstständigkeit - **Nachbesprechung**: Feedback und Verbesserungen #### **Regelmäßige Fortbildung** - **Monatliche Updates**: Neue Funktionen - **Quartals-Schulungen**: Vertiefung - **Jährliche Zertifizierung**: Kompetenz-Nachweis - **Spezialisierung**: Rolle-spezifische Fortbildung ## 11. Troubleshooting ### 11.1 Häufige Probleme #### **Anmeldung funktioniert nicht** 1. **Benutzername** prüfen (Groß-/Kleinschreibung) 2. **Passwort** prüfen (Sonderzeichen?) 3. **Caps Lock** prüfen 4. **Administrator** kontaktieren #### **System reagiert langsam** 1. **Netzwerk-Verbindung** prüfen 2. **Andere Programme** schließen 3. **System** neu starten 4. **IT-Support** kontaktieren #### **Drucker funktioniert nicht** 1. **Drucker-Status** prüfen 2. **Papier/Etiketten** kontrollieren 3. **Kabel-Verbindung** prüfen 4. **Drucker** neu starten ### 11.2 Notfall-Verfahren #### **System-Ausfall** 1. **Schichtführer** informieren 2. **Manuelle Verfahren** aktivieren 3. **Papier-Dokumentation** führen 4. **Nach System-Wiederherstellung** nachbuchen #### **Datenverlust** 1. **Sofort** melden 2. **Letzte Aktionen** dokumentieren 3. **Backup-Wiederherstellung** abwarten 4. **Fehlende Daten** nacherfassen ## 12.
